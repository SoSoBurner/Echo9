ECHO 9

Windows: double-click Play Echo 9.bat in this folder. It installs what it needs, builds the game, and opens it in your browser. Leave the black window open while playing.

Mac / Linux (or manual):

  • cd game
  • npm install (first time only)
  • npm run play

The game opens at http://localhost:4173. Your save persists between launches (browser storage).

Why not just open the game file directly? Browsers block the game's script modules when loaded straight off the disk, so it must be served locally — the launcher handles that.
